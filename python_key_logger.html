Python Key Logger

Key Logger Features

  • Code Snippet
    To the left, is a snippet of the code used to create this python key logger.
  • Objective
    The objective of this key logger was to gain a better understanding of how they work, how they are used in attacks and how to defend against them.
  • How This Key Logger Works
    This key logger uses the Python Keyboard library to capture a key as it is pressed. As a pressed key is captured, they are sent to a text document along with the exact time the key is pressed. The key logger has a kill command (kil) that when typed one after the other will stop the key logger and break the key capturing loop.
